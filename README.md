# Caculadora-JavaScript

Uma calculadora simples, feita usando HTML, CSS e JavaScrip.

Feita Baseada em diversos tutoriais e alguns cursos da Alura.

Tentei comentar o máximo possivel sem atrapalhar a leitura do código, para ter mais clareza na produção e demonstrar minha forma de raciocínio.

Esse projeto foi criado em 28/12/2021. Preferi registrar a data para posteriormente revisitá-lo e perceber as mudanças ocorridas em mim pela expêriencia e desenvolvimento técnico.
